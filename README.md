# Login-api

Micro Serviço para Cadastro de usuários.

## Tecnologias Utilizadas no Projeto
* Linguagem de programação: Java
* Framework: Spring boot
* Testes com MockMvc
* Banco de dados: H2 


## Comandos Importantes 

| Comandos  | Descrição |
|-----------|-----------|
| `mvn install`         | Instalação. |
| `mvn verify`          | Testes.     |
| `mvn spring-boot:run` | Running.    |
